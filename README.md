## 🖥️ Funcionalidades

### 📌 Automação Web
- Login automático
- Navegação entre páginas
- Preenchimento automático de formulários
- Cadastro automatizado
- Interações com botões e elementos HTML

### 📌 Localização de Elementos
Scripts focados em identificação de componentes da página:

- Seletores CSS
- XPath
- Localizadores Playwright
- Clique automatizado
- Manipulação DOM

### 📌 Web Scraping
- Coleta automática de informações
- Captura de links
- Extração de conteúdo

### 📌 Captura e Visualização
- Screenshot automática
- Visualização de páginas
- Gravação em vídeo da execução

## ⚙️ Instalação

Clone:

```bash
git clone https://github.com/matheusdiasz2007/ScriptsPW.git
```

Entre na pasta:

```bash
cd ScriptsPW
```

Crie ambiente virtual:

```bash
python -m venv venv
```

Ative:

Windows:

```bash
venv\Scripts\activate
```

Instale dependências:

```bash
pip install -r requirements.txt
```

Instale navegadores:

```bash
playwright install
```

## ▶️ Executando
